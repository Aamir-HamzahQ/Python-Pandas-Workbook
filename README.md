## 🧰 Key Python & Pandas Skills Demonstrated

### 🔍 Data Filtering & Subsetting
Used pandas to extract meaningful slices of data using:

- `.loc[]` for label‑based selection  
- `.iloc[]` for position‑based selection  
- Conditional filtering (e.g., `df[df["Sales"] > 500]`)  

These techniques were used to isolate customer segments, product categories, and high‑value transactions.

---

### 🧹 Data Cleaning & Manipulation

Applied essential pandas methods to prepare datasets for analysis:

- `dropna()` to remove missing values  
- `fillna()` to impute missing entries  
- `sort_values()` to order data by revenue, region, or product  
- `groupby()` to summarise sales by country, age group, or product type

### 📊 Data Visualisation

Created clear, insightful charts using:

- Pandas’ built‑in `.plot()`  
- `matplotlib.pyplot` for custom visualisations  

Visuals included:

- Bar charts for product performance  
- Line charts for sales trends  
- Pie charts for demographic distributions  

These helped reveal patterns such as top‑performing regions and customer behaviour trends.
